# Backend-Experiment3: User Login System

## Steps

### 1. Import Database
Use the same `user_registration_db` database and `users` table from Experiment 2.
Ensure passwords are hashed using PHP `password_hash()`.

### 2. Place Files
Put all files in:
MAMP: /Applications/MAMP/htdocs/experiment3

### 3. Run
MAMP:
http://localhost:8888/experiment3/login.html
